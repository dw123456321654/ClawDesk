## 项目简介

ClawDesk 是 OpenClaw 的桌面客户端，旨在降低 OpenClaw 的使用门槛，提供直观的服务管理和任务监控功能。

## 核心功能

- 🖥️ **服务管理**：一键启动/停止/重启 Gateway 服务
- 🌐 **全面中文化**：界面全中文，内置功能说明
- 🤖 **角色系统**：角色显示、手动切换、个性化配置
- 📊 **任务监控**：上下文监控、进度可视化、异常提示
- 💾 **任务持久化**：检查点机制、中断恢复、任务列表
- 💬 **会话管理**：多会话、历史记录、会话切换
- ⚡ **快捷操作**：健康检查、刷新状态、打开浏览器

## 技术栈

| 层级 | 技术选型 |
|------|---------|
| 桌面框架 | Tauri 2.x + Rust |
| 前端框架 | Vue 3 + TypeScript |
| UI 组件库 | Naive UI |
| 构建工具 | Vite |
| 状态管理 | Pinia |
